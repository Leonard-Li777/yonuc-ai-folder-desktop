# 虚拟目录说明

## 概述

`.VirtualDirectory` 是本应用自动生成的虚拟目录，用于展示经过智能整理后的文件结构。与原始目录中的文件保持一一对应关系，但采用了智能化的命名。

## 目的

此虚拟目录的主要目的是让用户能够预览文件整理的结果，而无需实际移动或复制原始文件。
当你对最终结果满意时，你可以一键点击“整理真实目录”，将真实目录也整理为与.VirtualDirectory一致的文件结构，然后本应用将会删除.VirtualDirectory目录。

## 技术原理

### 硬链接技术

虚拟目录中的文件采用硬链接技术生成。硬链接可以简单理解为文件的引用或别名，它具有以下特点：

1. 不占用额外的物理硬盘空间
2. 与原始文件共享相同的数据块
3. 对硬链接文件的修改会同步反映到原始文件
4. 删除硬链接文件不会影响原始文件
5. 删除原始文件，需要删除硬链接文件(本应用将主动嗅探真实目录的文件删除，对应删除虚拟目录中的硬链接文件。)

### 与快捷方式的区别

虽然硬链接在某些方面类似于快捷方式，但它们之间存在重要区别：

| 特性 | 快捷方式 | 硬链接 |
|------|---------|--------|
| 文件系统级别 | 仅Windows概念 | 操作系统文件系统特性 |
| 占用空间 | 极小(仅元数据) | 无额外空间 |
| 删除原始文件 | 快捷方式变为无效 | 硬链接仍可访问文件内容 |
| 修改内容 | 不影响原始文件 | 同步反映到所有链接 |
| 跨卷支持 | 支持 | 仅限同一文件系统 |
